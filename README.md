# Quantum Computing and Black Hole Scrambling

This repository explores scrambling of quantum information through random circuits and simple machine learning techniques. The project includes a short LaTeX paper, simulation scripts, and basic tests.

## Contents
- `paper/black_hole_scrambling.tex` – research paper describing the scrambling model. The figure uses the entropy plot generated by `random_scrambling.py`.
- `src/random_scrambling.py` – generates random circuits, reports entropy growth, optionally plots and runs on IBM hardware.
- `src/otoc.py` – computes a simple out-of-time-order correlator (OTOC) for small systems.
- `src/qnn_recovery.py` – demonstrates information recovery using a quantum neural network and reports fidelity.
- `tests/` – minimal tests for the fidelity routine.

## Installation
Install the Python requirements with
```bash
pip install -r requirements.txt
```

## Usage
### Scrambling Simulation
Run a scrambling simulation of 6 qubits and depth 20 and save the entropy plot:
```bash
python src/random_scrambling.py --qubits 6 --depth 20 --plot
```
Set `--backend BACKEND_NAME` to execute the circuit on an IBMQ backend after simulation. Counts are saved to `hardware_counts.png`.

### Entropy Visualization
The entropy values printed during the simulation are plotted to `entropy.png` when `--plot` is used. This plot is included in the LaTeX paper.

### OTOC Computation
Compute an OTOC for a two-qubit circuit of depth five:
```bash
python src/otoc.py --qubits 2 --depth 5
```

### QNN Information Recovery
Train a simple quantum neural network to recover the initial state from a scrambled state:
```bash
python src/qnn_recovery.py --qubits 2 --depth 3 --epochs 10
```
The script prints the final fidelity after training.

## Testing
Run the unit tests with
```bash
pytest
```

## Compiling the Paper
Generate the PDF of the paper with
```bash
pdflatex paper/black_hole_scrambling.tex
```
